# 📈 StockSnap AI

StockSnap AI is an AI-powered web application that analyzes and compares public stocks using real financial data and automated insights.

The tool helps users evaluate companies by examining revenue growth, profitability metrics, and price trends while generating beginner-friendly AI summaries.

---

## 🚀 Features

* 📊 Compare two stocks side-by-side
* 🔎 Analyze a single stock
* 📈 Interactive 6-month stock price charts
* 🤖 AI-generated investment summaries
* 🟢🟡🔴 Market Signal indicator (Bullish / Neutral / Risky)
* 💰 Financial metrics including:

  * Revenue
  * Net Income
  * Revenue Growth
  * Earnings Growth
  * Gross Margin
  * Operating Margin
  * P/E Ratio
* 🔥 Quick selector for top AI-related stocks

---

## 🛠 Technologies Used

* Python
* Streamlit
* Pandas

* OpenAI API
